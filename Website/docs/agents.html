Supported Agents

CLI Launcher manages 32+ terminal AI coding agents across three runtimes. Every entry installs, updates, and launches from the same panel.

The catalog is filterable inside the app by runtime. Below is the full set, grouped the same way. Each agent is installed through its real package — CLI Launcher resolves and pins it for you, so you never hand-edit a global environment.
