How It Works

01

Set Workspace

Select the local directory where your projects reside. The launcher configures target execution environments around this workspace location.

02

Manage Agents

Install, update, repair, or uninstall any supported coding client. The application issues NPM, Pip, or curl scripts in the background.

03

Launch and Code

Launch pre-configured terminal shells pointing directly to your projects. Start prompting your terminal assistant immediately.

Featured Assistants

Claude Code

Anthropic's terminal agent designed for direct file editing, workspace searching, test execution, and Git management within your repositories.

Aider

The premier command-line pairing assistant. Edit complex code across multiple files, discuss changes, and let it generate structured Git commits automatically.

Goose CLI

Block's open-source agentic assistant. Executes complete workflows, reads code context, runs compilation scripts, and tests files to solve engineering cards.

Gemini CLI

Google's terminal developer assistant. Seamlessly integrated for quick script writing, repository information fetching, and immediate terminal support.

Specifications

17 Unified Agents

Manage terminal AI tools like Claude Code, Gemini CLI, Aider, and Amazon Q all from a single control panel.

Lifecycle Actions

Install, update, repair, or uninstall agents dynamically. Direct platform execution with real-time feedback.

Configured Terminal

Instantly open shells (cmd, powershell, bash) automatically configured to run in your selected workspace path.

Workspace Reordering

Drag and drop agents to prioritize your favorites. Layout configuration automatically persists across sessions.

Version Badging

Automatic background queries to npm and pip alert you with update badges whenever a new CLI version releases.

Universal Installers

Native pre-packaged installer files for Windows (MSI, NSIS, Portable), macOS (DMG), and Linux (AppImage, deb, rpm).
